# Summer Learning Plan --- 2026

## 🎯 Goal

Build a portfolio that demonstrates real-world frontend development
skills, including UI/UX design, JavaScript proficiency, and deployment
workflows.

------------------------------------------------------------------------

## 🧭 Overall Path

Frontend-first → Real Projects → Backend Fundamentals

------------------------------------------------------------------------

## 🔥 Phase 1: Deep JavaScript (2--3 Weeks)

### Goals

-   Master core JavaScript concepts
-   Build confidence without relying on frameworks

### Topics

-   DOM manipulation
-   Event handling
-   Async JavaScript (`fetch`, `async/await`)
-   Arrays & objects (`map`, `filter`, `reduce`)
-   Modules (`import/export`)
-   Code structure & organization

### Project Ideas

-   Add dynamic filtering to music page
-   Improve audio player behavior
-   Add interactive UI states

------------------------------------------------------------------------

## ⚛️ Phase 2: React (3--4 Weeks)

### Goals

-   Learn modern frontend framework concepts
-   Understand component-based architecture

### Topics

-   Components
-   Props vs State
-   Hooks (`useState`, `useEffect`)
-   Conditional rendering
-   Lists & keys
-   Basic routing

### Project Ideas

-   Rebuild music page using React
-   Create reusable UI components

------------------------------------------------------------------------

## 🧱 Phase 3: Real Projects (3--4 Weeks)

### Goals

-   Build portfolio-quality projects
-   Focus on polish and usability

### Projects

1.  **Upgraded Portfolio**
    -   Improved UI/UX
    -   Smooth animations
    -   Refined layout and interactions
2.  **Main Project**
    -   Music dashboard (filter/search)
    -   Custom audio player UI
    -   Brand-focused web app

------------------------------------------------------------------------

## 🧠 Phase 4: Backend Fundamentals (2--3 Weeks)

### Goals

-   Understand fullstack basics
-   Connect frontend to backend

### Topics

-   Node.js basics
-   Express
-   APIs (requests/responses)
-   Connecting frontend to backend

### Project Ideas

-   Music catalog API
-   Contact form backend

------------------------------------------------------------------------

## ⚠️ What to Avoid

-   Learning too many frameworks
-   Jumping into backend too early
-   Staying in tutorial mode

------------------------------------------------------------------------

## 🧠 Framework Focus

-   Primary: React
-   Optional: Next.js (later)
-   Skip for now: Angular, Vue

------------------------------------------------------------------------

## 📅 Weekly Structure

-   4--5 days: Learn & build
-   1 day: Refactor & polish
-   1 day: Rest

------------------------------------------------------------------------

## 🚀 End Goal

A portfolio that demonstrates: - Real projects - Clean UI/UX - Strong
JavaScript skills - Deployment experience - Structured workflow

------------------------------------------------------------------------

## 💡 Mindset

Focus on building with what you learn, not just learning tools.
